# Steps to run this Code (Windows Only)

1. Download and install Conda Environment Manager
2. Download and install IsaacSim 5.1.0
3. Create a Conda environment and activate it
4. Download and install the necessary python packages using the requirements.txt
5. Run the model with this command:

	.\isaaclab.bat -p C:\IsaacLab\soccer\train.py --play --checkpoint .\model_4999.pt
